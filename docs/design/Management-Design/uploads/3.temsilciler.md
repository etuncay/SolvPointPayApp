## 2. Amaç ve Kapsam

Tüm temsilcilerin (acentelerin) listelendiği ve detayına gidildiği ana ekrandır. Temsilciler yalnızca **tüzel kişiliklerdir** ve kabulü yalnızca **Genel Müdürlük** üzerinden yapılır.

## 4. Menü ve Navigasyon

- Sol menüde "Temsilciler" ana başlığı; altında 3.1–3.4.
- Satıra tıklayınca `3.1 Yeni Temsilci` View/Update modunda açılır.

## 5. Ekran Yapısı ve Bileşenler

Tek panelli grid. **Kolonlar:** Temsilci No, Ticari Unvan, E-posta, Telefon, Vergi Kimlik No, Temsilci Grubu, Avans Hesap Bakiyesi (tek kolon: `TRY: … | USD: … | EUR: …`), Komisyon Mahsuplaşma (`settlement_frequency`), Oluşturma Tarihi, Son İşlem Tarihi, Status (`entity_status`, +gerekçe).

## 6. Alan Listesi

| Kolon | Tip | Aranabilir | Açıklama |
|---|---|---|---|
| Temsilci No | string | ✓ | Tekil; 99901+, Luhn |
| Ticari Unvan | string | ✓ | — |
| E-posta / Telefon | string | ✓ | Asıl iletişim kanalı |
| Vergi Kimlik No | string | ✓ | 10 hane |
| Temsilci Grubu | string | ✓ | Ücret/komisyon sınıflandırması |
| Avans Hesap Bakiyesi | composite | — | TRY/USD/EUR tek kolonda |
| Komisyon Mahsuplaşma | enum | ✓ | settlement_frequency |
| Oluşturma / Son İşlem Tarihi | date | ✓ (aralık) | — |
| Status | enum | ✓ | entity_status (+gerekçe) |

## 7. Validasyon Kuralları

Listeleme ekranıdır; filtre alanlarında tip/format kontrolü.

## 8. İş Kuralları ve Akışlar

- `record_status = 0` kayıtlar listelenmez.
- Avans Hesap Bakiyesi para birimi kırılımıyla tek kolonda gösterilir.
- Excel export mevcut filtreye göre çalışır.

## 9. Statü Modeli

`entity_status`: Active / Inactive / Blocked / Closed. KYC (KYB) akışı tüzel kişilerle aynı (bkz. `0.7`, `2.2`).

## 10. Veri Modeli

**`agent`** (özet): `agent_id (PK), trade_name, tax_no, agent_group_id, settlement_frequency, organization_type, registry_no, mersis_no, risk_segment, entity_status, entity_blockage_reason, entity_closure_reason, created_at/by, updated_at/by, record_status`

## 11. Servis / API Sözleşmeleri

| Servis | Method | Açıklama |
|---|---|---|
| `GET /agents` | GET | Filtre + pagination liste |
| `GET /agents/export` | GET | Excel |
| `GET /agents/{id}` | GET | Detay (3.1 ekranı) |
