# 7. İşlem Hareketleri

> Ortak kurallar için bkz. `0.genel-standartlar.md`.

## 1. Doküman Künyesi

| Alan | Değer |
|---|---|
| Menü | İşlem Hareketleri |
| Dosya | `7.islem-hareketleri.md` |
| Ekran tipi | Listeleme (grid) |
| İlgili ENUM | transaction_type, transaction_status |

## 2. Amaç ve Kapsam

Temsilcinin paydaşı olduğu (gönderen veya alıcı temsilcisi) tüm işlemlerin listelendiği ekrandır.

## 3. Aktörler ve Rol/Yetki Matrisi

| Aktör | List | View (Detay) |
|---|---|---|
| Temsilci Yetkili Kişi | ✓ | ✓ |

## 4. Menü ve Navigasyon

Sol menüde "İşlem Hareketleri". Satıra tıklayınca İşlem Onay (1.1) ekranı **Detay Modu**'nda, değiştirilemez ve butonsuz açılır.

## 5. Ekran Yapısı ve Bileşenler

Tek tablo. **Kolonlar:** İşlem Numarası, İşlem Tarihi, Gönderen Adı ve Soyadı/Unvanı, Alıcı Adı ve Soyadı/Unvanı, IBAN, İşlem Türü, Para Birimi, Tutar, Açıklama, Referans Numarası, Status, Temsilci Rolü (Gönderen Temsilcisi / Alıcı Temsilcisi), Toplam Gelir.

**Filtreler:** İşlem Numarası, İşlem Türü, İşlem Tarihi Aralığı, Tutar Aralığı (min–max), Gönderen, Alıcı, Temsilci Rolü.

## 6. Alan Listesi

| Kolon | Tip | Açıklama |
|---|---|---|
| İşlem Numarası / Tarihi | string / datetime | — |
| Gönderen / Alıcı Adı | string | — |
| IBAN | string | İlgili akışlarda dolu |
| İşlem Türü / Para Birimi / Tutar | — | — |
| Açıklama / Referans Numarası | string | — |
| Status | enum | transaction_status |
| Temsilci Rolü | enum | Gönderen / Alıcı Temsilcisi |
| Toplam Gelir | number | Temsilcinin o işlemden elde ettiği toplam gelir (TRY) |

## 7. Validasyon Kuralları

Salt-okunur listeleme ekranı; filtre tip/format kontrolü.

## 8. İş Kuralları ve Akışlar

- Varsayılan olarak **içinde bulunulan güne** ait işlemler gelir.
- Toplam Gelir, temsilcinin o işlemden elde ettiği toplam **TRY** cinsinden gelirdir.
- İşlem detayı İşlem Onay (1.1) sayfası Detay Modu'nda, değiştirilemez ve butonsuz açılır.

## 9. Statü Modeli

`transaction_status` — bkz. ortak para transferi statü modeli.

## 10. Veri Modeli

`transaction` — okuma; temsilcinin gönderen/alıcı temsilcisi olduğu kayıtlara filtreli.

## 11. Servis / API Sözleşmeleri

| Servis | Method | Açıklama |
|---|---|---|
| `GET /agent/transactions` | GET | Filtreli + pagination liste |
| `GET /agent/transactions/{id}` | GET | İşlem detay (1.1 Detay Modu) |
