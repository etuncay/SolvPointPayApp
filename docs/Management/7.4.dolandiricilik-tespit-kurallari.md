# 7.4 Dolandırıcılık Tespit Kuralları

> Ortak kurallar için bkz. `0.genel-standartlar.md`.

| Alan | Değer |
|---|---|

## 2. Amaç ve Kapsam

Fraud/işlem izleme kural motorunda tanımlı kuralların listelendiği ekrandır. Kural tanımlama/güncelleme/simülasyon işlemleri detay ekranında (7.4.1) yapılır.

## 3. Aktörler ve Rol/Yetki Matrisi

| Aktör | List | View (Detay) | Insert/Update |
|---|---|---|---|
| Uyum | ✓ | ✓ | ✓ |
| Risk | ✓ | ✓ | ✓ |

## 4. Menü ve Navigasyon

Risk ve Uyum menüsü altında. Satıra tıklayınca `7.4.1 Kural Tanımlama / Kural Detay` açılır.

## 5. Ekran Yapısı ve Bileşenler

Tek tablo. **Kolonlar:** Tetikleme Sırası, Kural Başlığı, Kapsam (`fraud_scope`), Status (`fraud_rule_status`), Öncelik (`level`), True Positive, False Positive, Açılan Vaka Sayısı, Ortalama Vaka İnceleme Süresi. Varsayılan sıralama: tetikleme sırasına göre artan.

## 6. Alan Listesi

| Kolon | Tip | Açıklama |
|---|---|---|
| Tetikleme Sırası | int | Kural çalışma sırası |
| Kural Başlığı | string | — |
| Kapsam | enum | Onboarding / Remittance |
| Status | enum | Active / Passive |
| Öncelik | enum | level |
| True / False Positive | int | Performans metrikleri |
| Açılan Vaka Sayısı | int | — |
| Ortalama Vaka İnceleme Süresi | süre | — |

## 7. Validasyon Kuralları

Listeleme ekranıdır; tanımlama validasyonu 7.4.1'de.

## 8. İş Kuralları ve Akışlar

- Kurallar **tetikleme sırasına göre yukarıdan aşağıya** işletilir; bir kural tetiklenirse sonrakiler çalışmaz (bkz. `0.11`).
- Performans metrikleri (TP/FP, vaka sayısı, ortalama süre) hesaplanmış alanlardır.
- Motorun mevzuat gereği asgari yüklenecek **seed kural seti** (T-006 serisi ve 19. madde) için bkz. `7.4.2.masak-kural-katalogu.md`; her kayıt `regulation_reference` ile ilgili kural koduna bağlanır.

## 9. Statü Modeli

`fraud_rule_status`: Active / Passive.

## 10. Veri Modeli

**`fraud_rule`**: `id, trigger_order, title, description, scope, condition_dsl, actions, status, priority, regulation_reference, version, created_at/by, updated_at/by, record_status`

## 11. Servis / API Sözleşmeleri

| Servis | Method | Açıklama |
|---|---|---|
| `GET /fraud-rules` | GET | Liste (tetikleme sırasına göre) |
| `GET /fraud-rules/{id}` | GET | Detay (7.4.1) |

## 12. Entegrasyonlar

Fraud kural motoru (runtime tüketici).
